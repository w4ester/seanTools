# Sound Assets

This directory contains sound effects for student interaction:

1. start.mp3 - Gentle beep for timer start
2. warning.mp3 - Friendly 30-second warning alert
3. complete.mp3 - Achievement completion sound
4. instruction.mp3 - Voice instruction guide

Note: Sound files should be:
- Short duration (1-3 seconds)
- Child-friendly tones
- Clear and distinct
- Not startling or too loud
